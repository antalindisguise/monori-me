# monori.me
![Build & Deploy](https://github.com/anthonymonori/monori-me/workflows/Build%20&%20Deploy/badge.svg?branch=master)

## Install

```Shell
gem install bundler
bundle install
```

## Build

```Shell
jekyll build --watch
```

## Deploy locally

```Shell
jekyll serve
```
